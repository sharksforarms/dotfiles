# NOTES

# Keymaps to remember

## Control mappings
ctrl+h - backspace
ctrl+u - delete till start of line
ctrl+w - delete previous word


## Incremental selection
gnn - start selection
grn - increase nodes selection
grm - decrease nodes selection
grc - increase scope selection

## Testing
t<C-n> - TestNearest
t<C-f> - TestFile
t<C-s> - TestSuite
t<C-l> - TestLast
t<C-g> - TestVisit
